2. Типичные (основные) симптомы депрессии 0/3

Для любой степени тяжести: необходимо как минимум 2 из 3. Для тяжелой: все 3.

3. Дополнительные симптомы депрессии 0

Каждая категория (обозначена жирным) засчитывается как 1 симптом. Общее количество (типичные + дополнительные): легкая ≥4 · умеренная ≥6 · тяжелая ≥8.
▸ Самооценка и уверенность в себе
▸ Беспричинное чувство самоосуждения / чрезмерная вина / самоуничижение
▸ Суицидальные мысли и поведение
▸ Снижение концентрации / нерешительность
▸ Нарушение психомоторной активности
▸ Нарушение сна любого типа
▸ Изменение аппетита с изменением веса
▸ Мрачное, пессимистическое видение будущего

4. Соматический синдром 0/4

Для квалификации соматического синдрома необходимо ≥4 из 8 симптомов. Пункт 5 (психомоторные нарушения) — это ОДИН симптом ("или").
